🚗 Automobile Sales Dashboard |

1.📊 Project Overview-
This project presents an interactive Automobile Sales Dashboard developed in Microsoft Excel to analyze and visualize key sales performance metrics across different countries, cities, and product categories. The dashboard transforms raw sales data into meaningful business insights, helping stakeholders track profitability, taxation, and sales trends effectively.

2.🚀 Key Features-
Total Profit Analysis
Total Tax Calculation & Monitoring
Year-wise Sales Performance Tracking
City-wise Sales Distribution Analysis
Country-wise Sales Comparison
Product-wise Profit & Tax Analysis
Interactive Filters and Slicers
Dynamic Charts and Visual Reports

3.🛠️ Tools & Skills Used-
Microsoft Excel
Pivot Tables & Pivot Charts
Data Cleaning & Transformation
Conditional Formatting
Dashboard Design
Data Visualization
Slicers & Interactive Reporting

4.📈 Insights Generated-
Identification of top-performing cities and countries
Analysis of overall business profitability
Product category performance evaluation
Sales trend monitoring across different regions
Tax contribution analysis for business reporting

5.🎯 Objective-
The objective of this project is to provide a centralized and interactive dashboard that enables users to monitor automobile sales performance, evaluate profit trends, and make data-driven business decisions efficiently.

6.📌 Dashboard Highlights-
✔ Professional Dashboard Design
✔ Interactive Data Exploration
✔ Real-Time Business Insights
✔ Easy-to-Understand Visualizations
✔ Performance Monitoring & Reporting
